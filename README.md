# 🎵 Saloosh – Music for the Soul

Free Indian Music App with 50M+ songs in Hindi, Punjabi, Tamil, Telugu & all Indian languages.

## ✨ Features
- 🎵 50M+ songs from JioSaavn
- 🌐 All Indian languages support
- 📱 PWA - Install on home screen
- 🎨 Premium UI/UX with animations
- 🔍 Search any song/artist
- ▶️ Full music player controls
- 📊 Progress bar with seeking
- 🔄 Auto-play next song
- 🚫 Zero Ads

## 🚀 Live Demo
[Visit Saloosh](https://your-link-here.netlify.app)

## 📱 Install on Phone
1. Open website in Chrome/Safari
2. Menu → "Add to Home Screen"
3. App installed! 🎉

## 🛠️ Tech Stack
- HTML5 + CSS3
- Vanilla JavaScript
- JioSaavn API (saavn.dev)
- Service Worker (PWA)
- Web Audio API
